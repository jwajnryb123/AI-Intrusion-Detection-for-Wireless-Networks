# AI-Intrusion-Detection-for-Wireless-Networks
Developed an Intrusion Detection System (IDS) using Artificial Intelligence to analyze wireless 802.11 data. Attacker data was gathered using a Kali Linux Virtual Machine (VM) to sniff wireless data across a network. Utilized Aircrack-NG to simulate wireless attack data. This data was then created and preprocessed wireless data using Kismet/Wireshark with pcap files. Supervised and reinforced learning AI models (Python) were compared and intergrated with Graylog.

Requirements:
- Kali Linux VM
- Personal Wirless Router (I used NetGear AX1800)
- Wireshark/Kismet
- Graylog (or any other open source SIEM)
